Requisitos:
- JDK 21
- IntelliJ IDEA
- Maven
- Visual Studio Code

Nota Importante:
Ao abrir o projeto no IntelliJ IDEA, certifique-se de que o Maven sincronize e baixe as dependências listadas no arquivo pom.xml. O IntelliJ geralmente faz isso automaticamente ou pergunta se você deseja importar as dependências do Maven.

Instruções de Execução:

1. Backend (IntelliJ):
   - Abra a pasta do projeto 'emergia_spring' no IntelliJ IDEA.
   - Aguarde o Maven carregar/baixar as dependências necessárias.
   - Encontre a classe 'EmergiaApplication.java' (localizada em src/main/java/br/com/emergia).
   - Execute a classe 'EmergiaApplication.java' para iniciar o servidor backend.

2. Frontend (Visual Studio Code):
   - Abra a pasta do projeto 'emergia_spring' no Visual Studio Code.
   - Navegue até o diretório 'src/main/resources/static'.
   - Abra o arquivo 'index.html' em seu navegador web (pode usar extensões como Live Server ou abrir o arquivo diretamente).

3. Uso da Aplicação:
   - No navegador, acesse a página da calculadora.
   - Preencha os campos com os dados necessários.
   - Aguarde o resultado ser exibido na tela.
   - Clique no botão 'Salvar Dados'.
   - Digite o nome da fazenda quando solicitado e confirme.
   - Após a página recarregar, clique em 'Visualizar resultado'.
   - Para consultar cálculos anteriores, clique em 'Visualizar histórico'.
